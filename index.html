Растровые и векторные изображения

Примеры и сравнение двух типов графики на веб-странице

В чем разница?

Растровые изображения состоят из пикселей (точек) и имеют фиксированный размер. При увеличении такого изображения качество ухудшается, появляются "квадратики" (пикселизация). Примеры форматов: JPG, PNG, GIF.

Векторные изображения состоят из математических формул, описывающих линии, кривые и формы. Их можно масштабировать без потери качества. Пример формата: SVG.

Растровые изображения (3 примера)

PNG изображение
Растровое

PNG (Portable Network Graphics) поддерживает прозрачность и сжатие без потерь. Идеально для логотипов и графики с прозрачным фоном.

JPG изображение
Растровое

JPG (Joint Photographic Experts Group) использует сжатие с потерями. Подходит для фотографий и изображений с множеством цветов.

Детальное растровое изображение
Растровое

При сильном увеличении растрового изображения становятся видны отдельные пиксели, что ухудшает качество картинки.

Векторные изображения (3 примера)

SVG иконка
Векторное

SVG (Scalable Vector Graphics) основан на XML. Иконка масштабируется без потери качества и имеет небольшой размер файла.

SVG графика
Векторное

Векторная графика идеально подходит для диаграмм, графиков и иллюстраций, которые нужно масштабировать.

Сложный векторный логотип
Векторное

Сложные векторные изображения могут содержать множество элементов, но остаются масштабируемыми без потери качества.
